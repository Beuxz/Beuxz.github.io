About the Project

I started this project because I couldn’t find a full Borderlands 2 map in Minecraft, so I decided to make one myself. I wanted to bring Pandora into Minecraft with attention to detail, scale, and the iconic art style. The map is being built following the main story progression for logical exploration.

Building Process

  • Time to complete a location: A few days to a few weeks (working solo)
  • Tools used: WorldEdit for bulk building and hand-building for detail
  • Art style: Sildur’s Shaders for a cel-shaded Borderlands look

Permissions: This map cannot be used, redistributed, or modified without explicit permission from the creator. Please contact me via GitHub or PlanetMinecraft before using it.
